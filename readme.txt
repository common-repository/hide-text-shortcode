== Description ==

= Hide Text Shortcode =

Useful shortcode to hide text in a post

[htsP type="Read More..."]Lorem ipsum dolor sit amet, consectetuer adipiscing elit.[/htsP].

This shortcode add a javascript function that  hide a text in a post and will show it on clicking on a choosen string.

This plugin is created by <a href="http://www.crawlermotori.com/" rel="friend" title="Danilo Franceschini">Danilo Franceschini</a>.


== Installation ==

1. Install Hide Text Shortcode.
2. Activate Hide Text Shortcode.
3. Add [htsP type="Read More..."]Lorem ipsum dolor sit amet, consectetuer adipiscing elit.[/htsP].
